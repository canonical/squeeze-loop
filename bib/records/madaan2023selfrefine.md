CLAIM CARDS

1. Quote (§1): "Self-Refine does not require any supervised training data, additional training, or reinforcement learning, and instead uses a single LLM as the generator, refiner and the feedback provider."
   Anchor: §1 Introduction.
   Paraphrase: A single frozen LLM plays all three roles - generator, feedback provider, and refiner - with no extra training.

2. Quote (§1): "The main idea is to generate an initial output using an LLM; then, the same LLM provides feedback for its output and uses it to refine itself, iteratively."
   Anchor: §1 Introduction.
   Paraphrase: The loop is closed entirely by the model's own critique: generate -> self-feedback -> self-refine -> repeat.

3. Quote (§2): "Given an initial output generated by a model M, we pass it back to the same model M to get feedback. Then, the feedback is passed back to the same model to refine the previously-generated draft."
   Anchor: §2 "Iterative Refinement with Self-Refine".
   Paraphrase: Formally the same model M is reused for feedback and refinement (Eqs. 2-3), no external oracle in the loop.

METHOD: Self-Refine prompts a single LLM to produce an output, then prompts the same model to give actionable feedback on that output, then to revise using the feedback, iterating until a stopping condition. It uses only few-shot prompts (no training, RL, or external supervision). Evaluated across 7 diverse generation/reasoning tasks (e.g. dialogue response, code optimization, math, sentiment reversal).

LIMITATIONS (authors' own): Requires a sufficiently capable base model (weaker models give poor feedback and little/no gain); relies on the model's ability to generate useful self-feedback, which is not guaranteed; mainly demonstrated on tasks where quality can be judged from the output itself; no external grounding so errors the model cannot detect persist.

CONTRIBUTION: Cited for "Self-Refine close[s] the loop with the model's own critique" with a single model as generator/feedback/refiner. Body states this verbatim (§1, §2). VERDICT: SUPPORTED.
