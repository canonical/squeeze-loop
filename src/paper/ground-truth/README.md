# ground-truth — the manuscript's lower bound (hard truth)

The executable substrate of the paper: every reported number is regenerated by a
harness, the manuscript compiles, and every citation resolves.

- `build_ground_truth.py` — re-run all macro generators (the single list lives in
  `../_paperlib.py`), rebuilding `tex/macros/*.tex` from the committed artifacts.
- `verify_ground_truth.py` — the hard checks: **determinism** (regenerated macros
  are byte-identical to committed, i.e. `git diff tex/macros` is empty), **build**
  (pdflatex green, all refs/cites/macros resolve; skipped honestly if pdflatex is
  absent), and **cites resolve** (every `\cite` has a reading record).

This is the lower bound an author's report cannot satisfy: the numbers either
recompute and the document either builds, or they do not.
