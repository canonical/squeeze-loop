# implementer (the writer)

The writer's in-band deliverable is the manuscript itself: `tex/paper.tex` (with
`tex/macros/` and `tex/references.bib`) at the repository root. It is intentionally
NOT copied here — the reflexive instance operates on the real paper, so the
deliverable lives where the paper lives. Every externally visible claim is tagged
`CLM-n` in `claims/ledger.tsv` and bound to a reading record or an artifact.
